# Bing Ads Java SDK

The Bing Ads Java Software Development Kit (SDK) Version 9.3 enhances the experience of developing Bing Ads applications with Java. The SDK includes a proxy to all Bing Ads API web services, provides a bulk file reader and writer, and abstracts low level details of authentication with OAuth. For more information, see [Bing Ads Java SDK](https://msdn.microsoft.com/en-US/library/bing-ads-java-sdk.aspx) on MSDN.

# Maven Artifact

```Xml
<dependency>
  <groupId>com.microsoft.bingads</groupId>
  <artifactId>microsoft.bingads</artifactId>
  <version>9.3.0-beta-1</version>
</dependency>
```
